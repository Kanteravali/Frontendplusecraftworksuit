API Documentation

Comprehensive documentation for all available API endpoints and their usage.

Authentication

All API requests require authentication using Bearer tokens. Include your API key in the Authorization header:

Authorization: Bearer your-api-key-here

User Management

Endpoints for managing user accounts and profiles.

GET /api/v1/users

Description: Retrieve a list of all users

Parameters: limit, offset, search

POST /api/v1/users

Description: Create a new user

Body: User object with name, email, role

Product Management

Endpoints for managing products in the catalog.

GET /api/v1/products

Description: Get all products with filtering options

PUT /api/v1/products/{id}

Description: Update a specific product
